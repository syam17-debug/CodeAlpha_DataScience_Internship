# CodeAlpha Data Science Internship

## Overview

This repository contains the projects completed during my Data Science Internship at CodeAlpha. These projects demonstrate practical applications of data analysis, visualization, machine learning, and predictive modeling using Python.

---

## Projects

### Task 1: Iris Flower Classification

**Objective:** Build a machine learning model to classify Iris flowers into different species.

**Key Features:**

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Random Forest Classification
* Accuracy evaluation
* Data visualization

**Skills Used:**
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn

---

### Task 2: Unemployment Analysis with Python

**Objective:** Analyze unemployment trends and study the impact of COVID-19 using data visualization techniques.

**Key Features:**

* Data cleaning
* Trend analysis
* COVID-19 impact analysis
* State-wise unemployment comparison
* Visual dashboards and graphs

**Skills Used:**
Python, Pandas, Matplotlib, Seaborn

---

### Task 3: Car Price Prediction

**Objective:** Predict the selling price of used cars using machine learning algorithms.

**Key Features:**

* Feature engineering
* Data preprocessing
* Model training and testing
* Price prediction
* Performance evaluation

**Skills Used:**
Python, Pandas, NumPy, Scikit-Learn, Matplotlib

---

### Task 4: Sales Prediction Using Python

**Objective:** Develop a predictive model to estimate future sales based on advertising and historical data.

**Key Features:**

* Data preprocessing and cleaning
* Correlation analysis
* Exploratory Data Analysis (EDA)
* Linear Regression Model
* Sales forecasting
* Model performance evaluation

**Skills Used:**
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
