# Sales Prediction using Python

## Overview

This project predicts product sales based on advertising expenditure across multiple media platforms such as TV, Radio, and Newspaper.

Machine Learning techniques are used to analyze the relationship between advertising investments and resulting sales, helping businesses make informed marketing decisions.

## Dataset

The dataset contains:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales (Target Variable)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Workflow

1. Load Dataset
2. Data Cleaning
3. Feature Selection
4. Train-Test Split
5. Linear Regression Model Training
6. Model Evaluation
7. Data Visualization

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Generated Visualizations

* Sales Distribution
* Actual vs Predicted Sales
* TV Advertising vs Sales

## Result

The Linear Regression model successfully predicts sales based on advertising expenditures and demonstrates the impact of marketing investments on business performance.
