# Unemployment Analysis with Python

## 📌 Project Overview

This project is completed as part of the CodeAlpha Data Science Internship.

The objective of this project is to analyze unemployment data using Python, identify unemployment trends, study the impact of COVID-19 on unemployment rates, and visualize important patterns through graphs and statistical analysis.

---

## 🎯 Objectives

* Analyze unemployment rate data.
* Perform data cleaning and preprocessing.
* Explore unemployment trends across different regions.
* Study the impact of COVID-19 on unemployment.
* Create meaningful visualizations.
* Generate insights that can support economic and social decision-making.

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

---

## 📂 Dataset

The dataset contains information related to:

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

---

## 🔍 Data Preprocessing

The following preprocessing steps were performed:

* Removed unnecessary spaces from column names.
* Converted date values into datetime format.
* Handled missing values using forward fill.
* Removed duplicate records.
* Prepared the dataset for analysis and visualization.

---

## 📊 Visualizations

The project includes the following visualizations:

### 1. Unemployment Trend Analysis

Shows how unemployment rates changed over time.

### 2. Region-wise Unemployment Analysis

Compares average unemployment rates across different regions.

### 3. Correlation Heatmap

Displays relationships between numerical variables in the dataset.

### 4. COVID-19 Impact Analysis

Illustrates the effect of the COVID-19 pandemic on unemployment rates.

---

## 📈 Key Insights

* Unemployment rates showed significant fluctuations over time.
* Certain regions experienced higher unemployment rates than others.
* The COVID-19 pandemic had a noticeable impact on employment levels.
* Labour participation rates and employment statistics provide additional insights into workforce dynamics.

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/syam17-debug/CodeAlpha_Unemployment_Analysis.git
```

2. Navigate to the project folder:

```bash
cd CodeAlpha_Unemployment_Analysis
```

3. Install required libraries:

```bash
pip install pandas matplotlib seaborn
```

4. Run the Python script:

```bash
python unemployment_analysis.py
```
